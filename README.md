# Stämme Online – Prototyp

Ein browserbasierter Prototyp im Stil von **Die Stämme**. Der Server liefert eine Single-Page-App mit Dorfverwaltung, Ressourcen, Handel, Angriffsplanung und einer 3D-inspirierten Weltkarte.

## Voraussetzungen

- Node.js (LTS empfohlen)

## Server starten

```bash
node server.js
```

Anschließend im Browser öffnen: `http://localhost:3000`.

## Health-Check

```bash
curl http://localhost:3000/health
```
